# Project structure

- `src/` — исходный код проекта.
- `tests/` — тесты.
- `docs/` — документация по проекту.
- `config/` — конфигурационные примеры и шаблоны.

Файлы:
- `README.md` — краткое описание и инструкция по запуску.
- `.gitignore` — базовые правила игнорирования служебных файлов.
- `src/main.py` — рабочая CLI-точка входа, загрузка конфига и генерация отчёта.
- `tests/test_smoke.py` — тесты загрузки конфига, ошибок, отчёта и CLI-режимов.
- `docs/structure.md` — описание структуры каталогов и файлов.
- `config/settings.example.json` — пример валидного JSON-конфига.
